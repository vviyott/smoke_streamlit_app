# 🚬 Smoke Streamlit App

서울시 흡연 현황과 민원 데이터를 시각화하고, 담배 관련 뉴스를 분석하는 AI 챗봇 시스템입니다.  
Streamlit 기반의 웹 앱으로 구성되어 있으며, 사용자는 다양한 시각 자료와 인터랙티브 기능을 통해 정보를 확인할 수 있습니다.

## 📌 주요 기능

- 📊 서울시 자치구별 흡연율 시각화 (Tableau 연동)
- 🗺️ 흡연구역 위치 지도 표시
- 📰 중앙일보 기사 기반 AI 뉴스 챗봇
- 🛒 네이버 쇼핑 API 기반 가격 비교
